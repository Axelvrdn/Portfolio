Application Mobile

Description du Projet

Ce projet consiste à développer une application mobile native pour les capitaines de la compagnie MarieTeam. L'application vise à simplifier la collecte et la transmission d'informations sur les traversées (état de la mer, retards, etc.), tout en gérant les contraintes de connectivité en mer. Les données saisies localement seront synchronisées avec le serveur principal dès que la connectivité sera rétablie.

Technologies Utilisées

  • Flutter (Framework multiplateforme)
  • Dart (Langage de programmation)
  • SQLite (Base de données locale)
  • API REST (Communication avec le serveur)
  • Git (Gestion de versions)
  • Adobe XD
    •

Fonctionnalités Principales

  • Saisie des informations sur les traversées (retards, état de la mer, etc.)
  • Enregistrement des données en local via SQLite
  • Synchronisation automatique avec le serveur lors de la récupération de la connectivité
  • Consultation en temps réel des données du serveur
  • Interface utilisateur optimisée pour une utilisation en mer (ergonomie simplifiée, mode hors-ligne)
