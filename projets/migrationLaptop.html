Migration Laptop

Description du Projet

Logiciel C# permettent d'identifier les erreurs de localisation associer aux laptop et identification de la date du leasing de manière plus intuidif et facile

Technologies Utilisées

  • C#
  • Github
  • Visio
  • Adobe XD

Fonctionnalités Principales

  • Récupère la date de leasing du laptop
  • Identifie les érreures de localisation associer aux laptop
  • Ne fonctionne pas avec une base de données pour éviter les problèmes de sécurité
